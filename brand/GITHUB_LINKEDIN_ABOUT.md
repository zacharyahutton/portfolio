## LinkedIn About — addition (paste into About section)

I build messaging bots and integration backends in Python — webhooks, conversational state, LLM routing, and production hardening on FastAPI. My live Telegram demo ([t.me/zachtedem_bot](https://t.me/zachtedem_bot)) covers multi-step booking flows, support tickets, 30+ FAQ topics, Groq/OpenAI chat with SQLite memory, rate limiting, and a Railway-deployed webhook server with a public health endpoint. The codebase is open source at github.com/zacharyahutton/telegram-bot-demo.

Alongside bot work, I deliver production full-stack software on contract: React/Next.js frontends, FastAPI backends, MongoDB, REST APIs, JWT auth, and deployment on Vercel and Railway. On weROI I shipped GrowthIQ, an LLM-powered chat assistant with quick-reply routing and FAQ fallbacks before model calls. Webhook Relay API is my sandbox for HMAC-signed outbound webhooks with exponential backoff — the same infrastructure patterns bots need when emitting events to CRMs or payment systems.

I am based in Jamaica, work remotely, and I am strongest where messaging platforms meet async Python backends. I read AI-generated code critically, refactor for edge cases, and ship integrations that fail loudly instead of silently. Open to part-time remote bot and integration work.
